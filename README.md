# WhatBytes Frontend Assignment

A product listing app built with Next.js and Tailwind CSS.

## Live Demo
[https://whatbytes-assignment-three-eta.vercel.app/](https://whatbytes-assignment-three-eta.vercel.app/)

## Tech Stack
- Next.js
- Tailwind CSS
- React Context API
- lucide-react

## Features
- Product listing with filters
- Category, price and brand filters
- Search functionality
- Product detail page
- Cart with quantity controls
- Persistent cart using localStorage
- Fully responsive design

## Run Locally
npm install
npm run dev
